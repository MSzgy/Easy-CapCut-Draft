Introduction

本课程是由Anthropic带来的agent skills 课程, skills是指一系列的instructions, agents可以根据这些指令进行更有效率的任务处理,可以加快速度,减少token的使用.

Why Use Skills - Part 1

在本节,从0讲解了如何定义一个skill, 以及skill 与MCP Agent之间的关系, 视频中举了相应的例子来组织各个部分.

Why Use Skills - Part 2

Skills vs Tools, MCP and Subagents

Exploring Pre-built Skills

在本节介绍了由anthropic 开发的几个skills

Powerpoint

skill-creator

可以用来创建想要的skill

Creating Custom Skills

我们可以利用skill- creator 去判断一个SKILL.md 的规范程度, 同时,我们也可以像写ut一样去衡量skill的运行效果.

Skills with the Claude API

视频中用代码举例,讲述了skill处理的步骤, 包括一开始如何选skill, 如何读文件,以及如何从skill.md中链接的文件去读相应的要求.

Skills with Claude Code

在这节视频, 演示了一个增加新命令行的skil是如何生效的.

简直是良心教学啊, 体验非常好. 任何基础操作都会提示的明明白白

我们可以在主agent继续执行任务的同时, 创建一个subagent 去review code.

Skills with the Claude Agent SDK

本节介绍了一个多agent 协同工作的demo, skill 作为主agent的guideline,引导子agen工作.

其中三个subagents可以并行工作,加快速度.
